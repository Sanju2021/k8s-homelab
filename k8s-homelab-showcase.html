Deployment

Zero to Cluster

Nine phases, fully scripted. Run ./scripts/setup.sh to execute all phases unattended.

01
Prepare Hyper-V Host
Enable WinRM on the Dell Server, create VHD storage directories, and download the Ubuntu 24.04 ISO.
# PowerShell (run as Administrator) winrm quickconfig -q New-Item -Path "D:\HyperV\VMs" -Type Directory
02
Configure Terraform
Copy the example vars file, fill in your server IP, NIC name, storage paths, and SSH public key.
cp terraform.tfvars.example terraform.tfvars # Edit: hyperv_host, ssh_public_key, # vhd_base_path, physical_adapter_name
03
Provision 16 VMs
Terraform creates all virtual machines, VHDs, and the external Hyper-V switch in one apply. ~15 minutes.
cd terraform/environments/production terraform init terraform apply
04
Base OS Hardening
Ansible hardens all 16 nodes simultaneously: swap disabled, kernel modules, sysctl tuning, UFW firewall, SSH.
ansible-playbook playbooks/site.yml \ --tags phase1 \ --vault-password-file .vault_pass
05
Bootstrap Kubernetes
Master-01 initialises the cluster with kubeadm. Masters 02‚Äì03 join as control-plane nodes. Workers join after.
ansible-playbook playbooks/site.yml \ --tags kubernetes \ --vault-password-file .vault_pass
06
Deploy Ceph Cluster
cephadm bootstraps on node-01, adds 4 more nodes, deploys 5 OSDs, creates the Kubernetes RBD pool.
ansible-playbook playbooks/site.yml \ --tags ceph \ --vault-password-file .vault_pass
07
Install Add-ons
Rook-Ceph operator, MetalLB (IP pool), and NGINX Ingress Controller are installed via Helm.
helm install rook-ceph rook-release/rook-ceph kubectl apply -f kubernetes/storage/ # MetalLB + Ingress auto-configured
08
Deploy Workloads
Apply all manifests: namespaces, StorageClass, 5 NGINX replicas, 5 PostgreSQL StatefulSet pods with PVCs.
kubectl apply -f kubernetes/namespaces/ kubectl apply -f kubernetes/workloads/webservers/ kubectl apply -f kubernetes/workloads/postgresql/
09
Deploy Monitoring
Prometheus, Grafana, Alertmanager, Node Exporter, kube-state-metrics, and PostgreSQL Exporter ‚Äî all wired up automatically.
kubectl apply -f kubernetes/monitoring/ # Access: grafana.homelab.local # Access: prometheus.homelab.local
